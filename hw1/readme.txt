This project is for basic operations on OpenCV. When you run the code, it ask a valid number to open the related image in the dataset. 
The 2x2 grid includes the original image, rotated image in the first two cell. When the user presed left button with mouse, the selected region 
will apper on the other cells.
To change the image, press "ENTER" when the curser is on the grid.
NOT: The dataset should be inside this folder and its name should be "Data". 


To run the code, first open the terminal on the build directory and then type:
cmake ..
then build the code by writing:
make
finally, run the main file which is an executable named as "main" so type:
./main
